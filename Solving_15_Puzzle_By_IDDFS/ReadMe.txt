The folder contain a IDDFS.java file which is implementation of Iterative Depth First Search Algorithm. Please execute that file by running it on eclipse or anyother IDE. The code will ask a input state of 15 puzzle. please type the state as continous characters with comma and space separation.
For example your input should be like this 
ip : 1, 0, 2, 4, 5, 7, 3, 8, 9, 6, 11, 12, 13, 10, 14, 15 if the state is 
	1  0  2  4
	5  7  3  8
	9  6  11 12
	13 10 14 15
The code will ouput a sequence of steps from given state to goal state and depth at which goal is found.
